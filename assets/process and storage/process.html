Process and Storage

Cooling Fan: Prevents overheating of components.
CPU Cooler: Cools down the processor for optimal performance.
Video Card: Renders graphics for display output.
RAM: Temporary memory for quick data access.
Power Supply: Provides power to all components.
Motherboard: Connects and allows communication between components.
CPU: The brain of the computer, processes instructions and tasks.
SSD: A storage device for faster data retrieval and storage.
